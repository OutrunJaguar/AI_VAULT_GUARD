# 🛡️ AI Vault Guard v2.0
**Private, Local-First AI Transcription & Summarization Engine**

### 🧠 The Problem
Cloud transcription services are expensive, compromise privacy, and require manual uploading. This project was built to automate the workflow securely on local hardware.

### ⚡ The Solution
A Python-based "Folder Watcher" that uses **OpenAI Whisper** for high-accuracy local transcription and **Gemma 3:12B** (via Ollama) for intelligent summarization. 

### ✨ Key Features
* **Zero-Cloud:** Everything stays on your local machine. No data leaks.
* **Ghost Mode:** Runs invisibly in the background via VBScript.
* **Automated Workflow:** Detects audio -> Transcribes -> Summarizes -> Archives.
* **Real-time Feedback:** Integrated Windows Desktop notifications via Plyer.
